# DESeq2
R Script for differential gene expression identification using featuresCount count matrix.

input: 
i) featuresCount count matrix (Tab-delimited .txt)
ii) Metadata table containing sample id and condition(s)

Output: 
i)   Dispersion plots
ii)  PCA plots
iii) MA plots
iv)  Volcano plots
v)   DESeq2 DEGs table
